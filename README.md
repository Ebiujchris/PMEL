# PMEL Advisory Centre Website

A modern, professional static website for PMEL Advisory Centre - a regional consultancy firm specializing in Programme Management, Monitoring & Evaluation, Research, and Organizational Development.

## 🎨 Design Features

### Color Palette
- **Primary Blue**: #1E3A8A (trust, professional)
- **Secondary Blue**: #3B82F6 (modern, tech-driven)
- **Accent Green**: #10B981 (growth, sustainability)
- **Accent Orange**: #F59E0B (call-to-action)
- **Dark Text**: #111827
- **Light Background**: #F9FAFB

### UI Principles
- Clean, corporate design
- Evidence-based visual hierarchy
- Responsive grid system
- Smooth animations and transitions
- Mobile-first approach
- Professional consultancy aesthetic

## 📁 File Structure

```
├── index.html          # Main HTML file
├── styles.css          # All styling and responsive design
├── script.js           # Interactive features and animations
└── README.md           # This file
```

## 🚀 How to Use

1. **Open the website**: Simply open `index.html` in any modern web browser
2. **No build process needed**: This is a static website - no installation required
3. **Edit content**: Modify `index.html` to update text and content
4. **Customize styles**: Edit `styles.css` to change colors, fonts, or layout
5. **Add features**: Extend `script.js` for additional interactivity

## 🔧 Customization Tips

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-blue: #1E3A8A;
    --secondary-blue: #3B82F6;
    --accent-green: #10B981;
    /* ... */
}
```

### Add Your Logo
Replace the text logo in `index.html`:
```html
<div class="logo">
    <img src="your-logo.png" alt="PMEL Advisory Centre">
</div>
```

### Update Content
All content is in `index.html` - simply find the section and edit the text.
